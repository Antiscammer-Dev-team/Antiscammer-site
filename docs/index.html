Getting Started

To invite AntiScammer to your server, use this link.

Setup

After inviting the bot, run:

/setup — This sets up default logging and permissions.

Core Commands

  • /setup — Initializes AntiScammer in your server
  • /report — Report a scammer
  • /falsepositive — Report a mistake or bad flag
  • /stats — View current stats

Logging & Alerts

Flagged messages are posted to your configured moderation log channel. Each message includes moderation buttons for false positive or escalation.
