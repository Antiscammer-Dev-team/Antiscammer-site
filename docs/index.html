Getting Started

To invite AntiScammer to your server, use this link.

Setup

Once the bot is in your server, run the command:

/setup

This will configure default logging channels and permissions.

Core Commands

  • /setup — Initializes bot in your server
  • /report — Manually report a user for scam behavior
  • /falsepositive — Flag a message or user as not a scam
  • /stats — Show bot usage statistics

Logging & Alerts

AntiScammer will automatically log flagged messages to your configured moderation channel. Each flagged message includes action buttons for moderators.
