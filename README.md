# 🚨 AntiScammer Bot

**AntiScammer** is a Discord bot designed to fight scams and protect communities by detecting scam messages, banning malicious users, and giving moderators powerful reporting tools.

> ✅ Visit the live site: [https://antiscammer-dev-team.github.io/Antiscammer-site/](https://antiscammer-dev-team.github.io/Antiscammer-site/)

---

## 🌐 Features

- 🔍 AI-based scam detection  
- 📊 Global stats and analytics  
- 🛡️ Global scammer ban sharing  
- 🧠 False positive reporting & human review  
- 🔔 Alert tools for moderators
